javaSE学习-java概述

发表于2023-06-02|更新于2023-06-06
|阅读量:

1.了解相关基础知识

  • GUI图形化界面

  • CLI命令行方式

  • Dos(Win+R) : 相关命令方式： md(创建目录)、cd(进入目录)、rd(删除目录)、del(删除文件,不能删除目录)

2.jdk相关知识

  • JDK=JRE+开发工具集
    JRE=JVM+JavaSE标准类库

  • HelloChina.java文件的编译并运行过程

    ①.HelloChina.java文件经javac.exe编译后生成HelloWorld.class字节码文件
    *命令语句: javac HelloChina.java
    ②.HelloWorld字节码文件经java.exe运行后输出运行结果
    *命令语句: java HelloWorld

  • 注意事项：

    ​ ①.一个class文件中可以有多个类，但不要同名，java区分大小写，所以编译可以通过，但只生成同名类的第二个类的字节码文件。

    ​ ②.public修饰符可以修饰类但只能修饰与文件名同名的类。

    ​ ③./** */注释为文档注释，进行javadoc -d mydoc -author -version HelloWord.java时，会被加载出来,且只能注释方法。

3.Java语言特性

  • 跨平台
  • 高性能
  • 简单性
  • 安全性
  • 易移植
  • 面向对象

4.企业常见问题

  • java程序还会出现内存溢出和内存泄漏问题吗？

    存在，内存泄漏：程序执行中有一块是不再使用的内存空间但JVM通过相关算法判断后不认为这是垃圾(即内存空间使用完毕之后未回收)

  • 怎么理解java是半编译半解释语言？

    java运行字节码文件(.class)有两种方式：①逐行翻译解释；②热点代码提前翻译成相关指令，在执行时直接使用机器指令执行(JIT)。
